# Main.py

Main.py is a program for finding filming locations near specific location.

## Installation needed

```bash
pip install argparse
pip install folium
pip install geopy
```

## Usage

```terminal

#creates html-file with map
python main.py main.py 2017 39.366426 -88.972351 locations.list

```
